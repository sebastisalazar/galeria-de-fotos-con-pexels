# Galería de Fotos con Pexels

Aplicacion web con funciones buscar, mostrar busquedas y guardar imágenes usando la API de Pexels.  

Desarrollado con HTML, CSS y JavaScript con una interfaz responsiva y funcional.

---

## Características

- Búsqueda de imágenes mediante el campo busqueda.  
- Visualización de resultados de busqueda o tendencia seleccionada en una galeria responsive.  
- Almacenamiento de fotos guardadas mediante LocalStorage como favoritas y visualizacion de ellas en  “Favoritos”  

---

## Tecnologías utilizadas

- HTML5  
- CSS3 (Flexbox + media queries)  
- JavaScript  
- API de Pexels  
- Visualizacion del despliegue en renders (https://proyecto-galeria-pexels.onrender.com/index.html)
